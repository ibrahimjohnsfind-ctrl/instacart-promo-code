Today’s Working Options

New users often see higher credits and free delivery. Existing users can still save with store specific promos, off peak delivery windows, and occasional credits that appear at checkout.

  • Free delivery offers show on eligible stores during checkout, watch the Service fees line for instant changes.
  • Category promos appear as Spend X, save Y. Add the required items, then check the applied discount before paying.
  • Instacart Plus free trial includes free delivery on many orders, try it if you place multiple orders per month.

How to Apply a Code

  1. Add items to your cart, meet any minimum shown by the store.
  2. Go to Checkout, tap Add promo or gift code.
  3. Paste the code, confirm, review the discount, then place the order.

Quick Fixes if a Code Fails

  • Remove any old codes, only one code may apply at a time.
  • Switch to a participating store if the current store is excluded.
  • Increase cart value to meet the minimum needed for the offer.
  • Try the desktop site if the app shows a temporary error.

FAQ

Do partner stores matter for codes?
Yes, some promos apply only to select partners. If the discount does not show, switch stores and test again.
Can I stack a store promo with free delivery?
Yes in many cases. Apply the store promo, choose a delivery window with lower fees, then check for free delivery messaging before you pay.
Does EBT SNAP affect promo eligibility?
Coupons may only apply to eligible items and fees. EBT covers approved groceries, any remaining balance and fees require a second payment method.
